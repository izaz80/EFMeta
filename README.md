# eFMeta — AI-Powered eFootball Meta Analysis

AI-generated tier lists and patch breakdowns for eFootball. Powered by Gemini AI. Free.

## Stack
- **Vercel** — hosting + serverless API proxy
- **GitHub Actions** — daily scraper (efootballhub.net)
- **Gemini 2.0 Flash** — AI analysis

## Setup

### 1. Deploy to Vercel
- Connect your GitHub repo to Vercel
- Add environment variable: `GEMINI_API_KEY = your_key_here`
- Deploy

### 2. Enable the Scraper
- Go to your GitHub repo → Settings → Actions → General
- Enable "Allow all actions"
- The scraper runs daily at 6am UTC automatically
- To run manually: Actions tab → "Scrape eFootballHub Players" → Run workflow
